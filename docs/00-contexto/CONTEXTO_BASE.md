# NeuroLearn

## Visão do Produto

O NeuroLearn não é uma plataforma de estudos.

O NeuroLearn é um **Sistema Operacional de Aprendizagem**.

Seu objetivo é transformar informação em memória de longo prazo e habilidade prática.

## Problema Resolvido

Hoje as pessoas:

- consomem conteúdo
- esquecem rapidamente
- não aplicam conhecimento
- não desenvolvem habilidades

O NeuroLearn existe para **reduzir o esquecimento** e **acelerar a aquisição de habilidades**.

## Base Científica

- Curva do Esquecimento de Ebbinghaus
- Pirâmide de Aprendizagem de Glasser
- Recuperação Ativa
- Aprendizagem Deliberada
- Metacognição

## Core Loop

```
Consumir → Recordar → Explicar → Aplicar → Revisar → Consolidar
```

## North Star Metric

**Quantidade de Habilidades Consolidadas por Usuário**

## Público-Alvo

- Estudantes

- Empresas que desejam acelerar aprendizagem
