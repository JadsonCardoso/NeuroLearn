# VISÃO

O NeuroLearn é um Sistema Operacional de Aprendizagem.

Não é:

- biblioteca digital
- aplicativo de notas
- plataforma de cursos
- sistema de flashcards

É uma plataforma que transforma informação em memória de longo prazo e habilidade prática.

---

# MISSÃO

Reduzir o esquecimento.

Aumentar retenção.

Aumentar aplicação prática.

Acelerar aquisição de habilidades.

---

# PROBLEMA

Atualmente pessoas:

- consomem conteúdo
- esquecem rapidamente
- não revisam corretamente
- não aplicam conhecimento
- não conseguem medir evolução

O NeuroLearn resolve esse problema.
